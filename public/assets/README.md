Asset folders for local media files.

Logos:
- Put organization logos in `public/assets/logos/organizations/`
- Recommended format: transparent PNG or SVG
- Recommended size: 256x256+
- File naming (current data mapping):
  - `mcmaster-ai-society.png`
  - `mcmaster-mix-club.png`
  - `studyplus-education-me.png`
  - `campus-sutra.png`
  - `gocode-academy.png`
  - `prestige-computer-services.png`

Project images:
- Put project thumbnails/cover images in `public/assets/projects/`
- Suggested subfolders are already created per project ID.
- Suggested cover image name per folder: `cover.png`
- Recommended size: 1600x900 (or 1200x675), optimized web PNG/JPG/WebP.

Usage:
- Any file inside `public/` is served from `/`.
- Example: `public/assets/logos/organizations/mcmaster-ai-society.png`
  becomes `/assets/logos/organizations/mcmaster-ai-society.png`.
